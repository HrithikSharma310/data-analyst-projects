Project: Sales Performance Dashboard

Business Problem:
Analyze sales data to track revenue, profit, customers, and orders.

Tools Used:
- Excel
- SQL (PostgreSQL)
- Power BI

KPIs:
- Total Revenue
- Total Profit
- Average Order Value
- Customer Count
- Month-over-Month Growth

Key Insights:
- Top 20% customers generated majority of revenue
- Certain regions consistently underperformed
